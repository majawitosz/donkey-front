# System zarządzania personelem w branży gastronomicznej
## Aplikacja webowa (Next.js)

Projekt został zrealizowany przy użyciu frameworka [Next.js](https://nextjs.org)
i utworzony za pomocą narzędzia create-next-app.

## Opis projektu

Celem projektu jest stworzenie aplikacji webowej wspierającej zarządzanie personelem
w lokalach gastronomicznych. System przeznaczony jest dla pracowników oraz kadry
zarządzającej i umożliwia usprawnienie procesów organizacyjnych.

## Wymagania systemowe

Aby uruchomić projekt lokalnie, wymagane jest posiadanie:

- Node.js (zalecana wersja LTS)
- menedżer pakietów:
  - npm (instalowany razem z Node.js)
  - lub yarn
  - lub pnpm
- przeglądarka internetowa (np. Chrome, Firefox, Edge)

    
## Uruchomienie projektu

### Instalacja zależności

```bash
npm install
```

### Uruchomienie serwera developerskiego

```bash
npm run dev
```

Alternatywnie:

```bash
yarn run dev  
pnpm run dev  
```

## Dostęp do aplikacji

Po uruchomieniu serwera aplikacja będzie dostępna pod adresem:

http://localhost:3000
